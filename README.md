# Canva App — Panduan Singkat (Bahasa Indonesia)

Clone dulu

## 1. Install

1. Pastikan Node.js versi 24
2. Pasang dependensi:

```bash
npm install
```

## 2. Cara Pake

1. Run Server

```bash
npm start
```

1. Server di `http://localhost:8080`.
2. Untuk melihat aplikasi di editor Canva, gunakan fitur **Preview** di Developer Portal (lihat langkah berikutnya).

## 3. Tambahkan (Register) App di Canva Developer Portal

1. Buka: https://www.canva.com/developers/apps dan buat aplikasi baru.
2. Di pengaturan app, pilih **App source > Development URL**.
3. Masukkan URL pengembangan, mis. `http://localhost:8080` (atau `https://localhost:8080` jika memakai HTTPS).
4. Klik **Preview** untuk membuka editor Canva yang memuat app Anda.
